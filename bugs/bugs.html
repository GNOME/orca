Orca Bug Dependency List

This page contains a list of bugs filed against external components and applications. This allows us to better track which releases of external components that Orca will work with and also the behavior one might experience in the presence of the bug.

  • GNOME Bugs
  • Evolution Bugs
  • Firefox Bugs
  • OpenOffice Bugs

GNOME Bug List

Note that you can do a GNOME Bugzilla Database Search for Orca-Related Bugs

Where to file GNOME bugs:

  • URL: http://bugzilla.gnome.org/enter_bug.cgi
  • CC: gnome-access-bugs@basso.sfbay.sun.com
  • Keywords: "accessibility"
  • NOTE: ATK is the application side of things, GAIL provides the ATK implementation for GTK, AT-SPI is the assistive technology side of things and also provides the atk-bridge and AT-SPI registry. When entering a bug, give some thought into where the bug might lie.
